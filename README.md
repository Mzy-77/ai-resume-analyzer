
AI Resume Analyzer

A modern, intelligent resume analysis app built with React, React Router, and Tailwind CSS that helps job seekers get instant insights and feedback on their resumes. This app scans uploaded resumes and uses core logic (and optionally AI) to deliver structured results and score formats.

🧠 Features

✔️ Clean, responsive UI powered by Tailwind CSS
✔️ Client-side routing with React Router
✔️ Upload and view resume files
✔️ (Optional) AI-based feedback & scoring
✔️ Fast local dev with Vite + React Router config

🛠️ Tech Stack
Technology	Purpose
React	Frontend UI
React Router	Navigation & routing
Tailwind CSS	Styling utilities
Vite	Fast build tool & dev server
Zustand (optional)	State management
pdfjs-dist	Resume PDF handling
